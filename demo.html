How It Works

This demo shows how to use the Cloudflare Worker proxy to access R2 storage securely:

  • Public files: https://your-worker.your-domain.workers.dev/object-key
  • Protected files: https://your-worker.your-domain.workers.dev/protected/object-key?secret=your-secret

The worker provides rate limiting, security headers, and prevents direct R2 link exposure.
